# Retail Loyalty Program Usage Analysis

This project performs an analysis of a retail loyalty program using **Python and Pandas**.  
The dataset is created directly inside the Python file (no CSV required) and is used to analyze customer usage patterns and retention insights.

---

## Objective
- Analyze loyalty program usage
- Understand customer retention behavior
- Compare active vs inactive customers
- Visualize insights using bar charts

---

## Tools & Libraries Used
- Python 3
- Pandas
- Matplotlib

---

## Dataset Description
The dataset is created as a Pandas DataFrame with the following columns:

- customer_id – Unique customer identifier  
- visits – Number of store visits  
- total_spent` – Total amount spent by the customer  
- points_redeemed – Loyalty points redeemed  
- active – Customer status (Yes / No)

---

## Features Implemented
- DataFrame creation without CSV files
- Descriptive statistics using Pandas
- Customer retention analysis
- Group-wise analysis (Active vs Inactive)
- Bar chart visualizations
- Correlation analysis

---

## Output & Insights
- Active loyalty members visit more frequently
- Active members spend more on average
- Higher loyalty engagement improves customer retention
